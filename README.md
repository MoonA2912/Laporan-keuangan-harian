# Aplikasi Laporan Keuangan Harian

## Deskripsi
Sebuah aplikasi sederhana yang bisa mencatat laporan keuangan harian. Aplikasi ini berguna untuk memasukkan pemasukan dan pengeluaran, dan menghitung saldo keuangan.
